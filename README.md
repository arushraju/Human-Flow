# Human-Flow
This is my work in the field of Crowd Dynamics. It's an attempt to replicate the result the Hughe's Pedestrian flow and use it to simulate Crowd management problems inspired from Indian Temples. The solver uses Finite Difference Method to solve the Partial Differential Equations.
YOu can read about it more on this <a href="https://arushraju.github.io/Human-Flow/">GitHub Page</a>
